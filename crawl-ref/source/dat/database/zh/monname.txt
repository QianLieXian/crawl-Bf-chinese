# monname.txt: 查找怪物名称
#
 # 该文件包含怪物名称的名称列表。
 # 目前仅当您的兽人使徒之一为
 # Beogh 的弥赛亚被 Beogh 命名时才使用，以及 Hep 的祖先名称。
 # 双持鳕鱼物种的武器名称也来自
 # 这里。
#
 # 一些重要的兽人音节
 # beogh, bog = 指的是 Beogh
 # ork, orc, org, og, ok 等 = 指的是兽人
#
 # 其他音节可能是从现实生活中借用的，或者是自己编造的。
#
 # 要确定兽人使徒的名称，游戏会查找该
 # 使徒的具体类型，后跟关键字“名称”，例如“兽人
#使徒战士名字”或者“兽人使徒巫师名字”。使徒可以获得
 # 一个特定类型独有的名字或一个普通的兽人名字。
#
 # 怪物名称支持怪物语音功能的一小部分：
 # @RANDGEN@、@random_god@、@random_god_chaotic@、@random_god_evil@、
 # @random_god_good@、@random_skill@、@random_skill_magic@ 和
 # @random_skill_mundane@。
#
 # 请随意添加到此列表中。与其他数据库列表一样，
 # 您可以添加权重（默认为 w:10）以使某些名称更可能出现
 # 或更少。
#
##########################################################
